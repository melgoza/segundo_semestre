Menú de navegación

¿Qué es un menú de navegación?

Es la parte de una página web donde, si haces click te manda a otras páginas web.

¿Cómo funciona un menú de navegación?

Usaun hipervínculo con el enlace a la página que quieres que visiten. Se usa la etiqueta "nav" para indicar que es el menú de navegación, y la etiqueta "a" para poner un hipervínculo, y si quieres poner el menú en lista, debes usar las etiquetas "ol" o "ul" y "li"
